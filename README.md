## 📝 الوصف
تطبيق دردشة متكامل يستخدم أحدث نماذج الذكاء الاصطناعي مفتوحة المصدر. يوفر واجهة عربية أنيقة وسهلة الاستخدام، ويدعم عدة نماذج للاختيار بينها.

## ✨ المميزات
- 🎨 واجهة مستخدم احترافية باللغة العربية
- 🧠 دعم ثلاث نماذج مختلفة: GPT-2، DialoGPT، Flan-T5
- ⚡ استجابة سريعة وذكية
- 🔒 خصوصية تامة - لا نخزن محادثاتك
- 🎛️ إعدادات متقدمة للتحكم في الردود
- 📱 متوافق مع جميع الأجهزة (جوال، تابلت، كمبيوتر)

## 🚀 طريقة الاستخدام
1. افتح التطبيق من الرابط المباشر
2. اختر النموذج المناسب من القائمة الجانبية
3. اكتب سؤالك في حقل الدردشة
4. انتظر الرد الفوري!
